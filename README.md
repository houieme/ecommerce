

## Quick Start



```bash
# Install server dependencies
npm install

# Install front dependencies
cd front
npm install

# Run both Express & React from root
npm run dev
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

# Build for production
cd front
npm run build
```
